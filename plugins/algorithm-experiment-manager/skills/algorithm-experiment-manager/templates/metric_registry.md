# 通用指标注册表

> 根据实验类型选择相关指标，不要求所有指标都出现。未提供的指标写“待补充”，不适用的指标写“不适用”。

## 通用机器学习指标

| 指标 | 含义或用途 | 记录要求 |
| --- | --- | --- |
| MAE | 平均绝对误差 | 记录整体值及关键子集值 |
| MSE | 均方误差 | 说明是否受异常值影响 |
| RMSE | 均方根误差 | 与目标量纲一致，便于解释 |
| R² | 拟合优度 | 记录测试集或验证集来源 |
| MAPE | 平均绝对百分比误差 | 说明零值或小值处理方式 |
| Accuracy | 分类准确率 | 类别不平衡时需配合其他指标 |
| Precision | 精确率 | 记录正类定义 |
| Recall | 召回率 | 记录漏检或漏判代价 |
| F1 | Precision 与 Recall 的综合指标 | 说明宏平均、微平均或加权平均 |
| AUC | 分类排序能力 | 记录 ROC-AUC 或 PR-AUC |

## 深度学习训练指标

| 指标 | 含义或用途 | 记录要求 |
| --- | --- | --- |
| training loss | 训练损失 | 记录最终值和曲线趋势 |
| validation loss | 验证损失 | 记录最佳值、最终值和过拟合迹象 |
| learning rate | 学习率 | 记录初值、调度器和变化曲线 |
| epoch | 训练轮数 | 记录总轮数和有效轮数 |
| early stopping epoch | 早停轮数 | 记录早停规则 |
| best checkpoint | 最优权重文件 | 记录路径和选择标准 |
| 训练时间 | 计算成本 | 记录硬件和总耗时 |
| 显存占用 | 资源使用 | 记录峰值和 batch size |

## 图像检测指标

| 指标 | 含义或用途 | 记录要求 |
| --- | --- | --- |
| Precision | 检测精确率 | 记录 IoU 阈值 |
| Recall | 检测召回率 | 记录漏检类别 |
| mAP@0.5 | IoU=0.5 下平均精度 | 记录验证集或测试集 |
| mAP@0.5:0.95 | 多 IoU 阈值平均精度 | 记录评估脚本版本 |
| 类别AP | 各类别 AP | 记录类别间差异 |
| 漏检率 | 未检出目标比例 | 记录目标尺度和场景 |
| 误检率 | 错误检测比例 | 记录常见误检来源 |
| 推理速度 | FPS 或单图耗时 | 记录硬件、输入尺寸、batch size |

## 时序预测指标

| 指标 | 含义或用途 | 记录要求 |
| --- | --- | --- |
| 不同预测时距 MAE | 多步预测绝对误差 | 分别记录各预测时距 |
| 不同预测时距 RMSE | 多步预测均方根误差 | 分别记录各预测时距 |
| 不同预测时距 R² | 多步预测拟合优度 | 分别记录各预测时距 |
| 峰值误差 | 峰值大小偏差 | 说明峰值定义 |
| 峰现时间误差 | 峰值发生时间偏差 | 记录时间分辨率 |
| FAR | False Alarm Ratio | 说明事件阈值 |
| POD | Probability of Detection | 说明事件阈值 |
| CSI | Critical Success Index | 说明事件阈值 |

## 可靠度分析指标

| 指标 | 含义或用途 | 记录要求 |
| --- | --- | --- |
| 失效概率 Pf | 系统或构件失效概率 | 记录估计方法和置信度 |
| 可靠指标 β | 可靠度指标 | 记录计算公式 |
| Subset Simulation 层级数 | 子集模拟复杂度 | 记录每层阈值 |
| 变异系数 | 估计稳定性 | 记录样本量 |
| 低概率区间误差 | 小失效概率精度 | 说明参考值来源 |
| FOS误差 | 安全系数误差 | 记录基准算法 |
| 失效模式一致性 | 物理或工程合理性 | 记录失效区域或模式 |

## 参数反演指标

| 指标 | 含义或用途 | 记录要求 |
| --- | --- | --- |
| 参数RMSE | 反演参数误差 | 记录真实值或参考值来源 |
| 后验均值误差 | 后验均值偏差 | 记录先验设置 |
| 后验覆盖率 | 置信区间覆盖能力 | 记录置信水平 |
| 后验收缩程度 | 不确定性降低程度 | 与先验方差对比 |
| 观测拟合误差 | 模型输出与观测差异 | 记录观测噪声假设 |
| 参数可辨识性 | 参数是否可稳定识别 | 记录敏感性或相关性分析 |

## 有限元或数值模拟指标

| 指标 | 含义或用途 | 记录要求 |
| --- | --- | --- |
| 收敛步数 | 非线性或迭代求解收敛情况 | 记录收敛准则 |
| 残差范数 | 数值收敛误差 | 记录最终值和曲线 |
| 时间步长 | 时间离散设置 | 记录自适应规则 |
| 质量守恒误差 | 守恒性质 | 记录计算区域和归一化方式 |
| 能量误差 | 物理一致性 | 记录能量定义 |
| 边界通量 | 边界条件影响 | 记录边界名称和方向 |
| 关键场变量 | 位移、应力、孔压、温度等 | 记录位置、时刻和单位 |
| 物理现象一致性 | 与工程认知或理论是否一致 | 记录异常现象 |

## 优化算法指标

| 指标 | 含义或用途 | 记录要求 |
| --- | --- | --- |
| 目标函数值 | 优化目标表现 | 记录最优值和最终值 |
| 约束违反程度 | 可行性 | 记录最大违反值和违反约束 |
| 迭代次数 | 收敛成本 | 记录停止条件 |
| 收敛曲线 | 搜索过程 | 保存图表路径 |
| 最优解稳定性 | 多次运行一致性 | 记录随机种子和方差 |
| 计算成本 | 时间、函数调用、内存等 | 记录硬件和并行设置 |
| 多次运行方差 | 随机算法稳定性 | 记录运行次数 |
